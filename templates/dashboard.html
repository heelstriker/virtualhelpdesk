VirtualHelpDesk

A lightweight Endpoint Management & IT Operations platform built with Python, Flask, SQLite, and PowerShell automation.

VirtualHelpDesk is a web-based IT operations and endpoint management system designed to simulate real-world enterprise support environments. It centralizes endpoint visibility, compliance monitoring, and IT asset tracking into a unified dashboard for desktop support and IT operations teams.

Overview

This project demonstrates how enterprise IT operations can be modeled using modern web technologies and automation scripting. It combines backend data management with frontend dashboards and PowerShell-based endpoint simulation.

The goal is to replicate a realistic IT support environment including inventory management, compliance tracking, and operational workflows.

Key Objectives

  • Endpoint inventory management
  • Software compliance monitoring
  • Hardware asset tracking
  • Patch status validation
  • Department-based resource organization
  • Help desk workflow simulation and automation

Enterprise Concepts Demonstrated

  • Configuration Management Database (CMDB)
  • Asset lifecycle tracking
  • Endpoint compliance monitoring
  • Software deployment workflows
  • Role-based access control concepts
  • IT operations dashboard design
  • Department-aware infrastructure modeling

Devices are organized using hostname conventions, IP addressing, and subnet allocation to simulate departmental ownership and network segmentation.

Example

LAXACTPC002 Accounting Department 17.110.120.2 LAXITPC002 IT Department 17.110.121.2

Technology Stack

Python (Flask, Jinja2) SQLite HTML / CSS PowerShell GitHub Render Deployment

Notes

This project is intended for educational and demonstration purposes, showcasing how endpoint management and IT operations concepts can be implemented in a lightweight web-based system.
