PowerShell Command Library

Frequently used Windows PowerShell commands for IT Help Desk and Desktop Support.

🖥️ System Inventory

Command Purpose Virtual Help Desk Simulation
Get-ComputerInfo Display computer hardware and OS information. Show device information from the Devices table.
Get-WmiObject Retrieve Windows system information using WMI. Display BIOS, CPU, RAM, and manufacturer details.
Get-Process List running processes. Display simulated running applications.
Display Windows services. Show service status and simulate stopped services.
Get-HotFix Display installed Windows Updates. Read installed patches from Patch table.

🌐 Network Diagnostics

Command Purpose Virtual Help Desk Simulation
Test-Connection Ping a remote host. Verify whether servers are online.
ipconfig Display IP configuration. Show device IP, subnet, gateway, and DNS.
Resolve-DnsName Resolve DNS records. Verify hostname resolution.
nslookup Query DNS server. Display DNS lookup results.
Get-NetAdapter Display network adapters. Show adapter status and simulate cable unplugged.
arp -a Display ARP cache. View neighboring devices on the subnet.
route print Display routing table. View simulated routing information.

💾 Storage & File Shares

Command Purpose Virtual Help Desk Simulation
Display mapped drives. Read Network Drives table.
Get-SmbMapping Display SMB mapped drives. Show mapped network drives and connection status.
Get-SmbShare Display shared folders. Show shared folders hosted on servers.
Get-Volume Display disk volumes. View storage utilization.
Get-Disk Display physical disks. View installed storage devices.
Get-Partition Display disk partitions. Show logical partition information.

🖨️ Printer Management

Command Purpose Virtual Help Desk Simulation
Display installed printers. Read Printers table.

📦 Software Management

Command Purpose Virtual Help Desk Simulation
Get-Package Display installed software. Read Software table.
Get-AppxPackage Display installed Microsoft Store apps. Show Windows App packages.
winget list Display installed applications. View installed software inventory.
winget search Search available software. Search Software Catalog.
winget install Install applications. Simulate software deployment.
winget upgrade Upgrade installed applications. Simulate software updates.
winget uninstall Remove installed applications. Simulate software removal.
Start-Process msiexec.exe Install MSI packages silently. Deploy enterprise MSI packages.
Start-Process setup.exe Run EXE installers silently. Deploy legacy software.

📋 Event Logs

Command Purpose Virtual Help Desk Simulation
Get-EventLog Display Windows Event Logs. View simulated system events.
Get-WinEvent Display advanced Event Logs. Investigate simulated incidents.

👤 User Management

Command Purpose Virtual Help Desk Simulation
Get-LocalUser Display local user accounts. Show users assigned to devices.
Get-LocalGroup Display local groups. View administrator and security groups.
Get-LocalGroupMember Display group members. View user memberships.
